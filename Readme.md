## 🚀 What’s inside

A collection of C++ programs covering fundamental DSA concepts:

* Arrays & Vectors
* Binary Search
* Sorting Algorithms (Merge Sort, DNF Sort, etc.)
* String Manipulation
* Recursion Basics
* Problem Solving Patterns
* STL Usage

## 📂 File Overview

Some of the key files included:

* `array.cpp` – Basic array operations
* `binarysearch.cpp` – Binary search implementations
* `mergesort.cpp` – Merge sort algorithm
* `sortingalgo.cpp` – Various sorting techniques
* `containerwater.cpp` – Two-pointer problem
* `rotatedsortedarray.cpp` – Modified binary search
* `palindrome.cpp` – String/number palindrome check
* `vector.cpp` – STL vector usage
* `stllibraries.cpp` – STL exploration
